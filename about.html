About this wiki

The Arcane Archive is a comprehensive, text-only fan reference for the co-op wave-survival magic-brawler The Spell Brigade by BoltBlasterGames. It is built and maintained by players.

What this is

A static, search-indexed HTML reference for every spell, hero, artifact, enemy, boss, world, wave, mechanic, curse, modifier, upgrade, and challenge currently shipped in the game. No accounts, no comments, no trackers, no JavaScript build pipeline — just plain HTML, CSS, and one small JS file (js/wiki.js) for progressive enhancements like search and sortable tables.

Sources

All factual content on this wiki is derived from the shipped game files of The Spell Brigade. Numbers like spell damage, artifact parameters, wave compositions, and enemy stats are read from the publicly distributed game data and rendered here for reference.

The wiki ships no game art, audio, sprites, or trademarked assets. Only factual numeric data and synthesized text descriptions are reproduced, organized into a navigable reference.

Methodology

Game data is organized into a set of JSON files (one per entity family: spells, artifacts, characters, enemies, bosses, waves, mechanics, meta). Each page on this wiki is rendered from those JSON files using plain HTML templates, with cross-reviewers checking numeric accuracy against the source.

Where a mechanic could not be confirmed with certainty, the page flags it with an inline note. The site contract — URL conventions, anchor patterns, template names, cross-linking rules — is documented and treated as a hard contract by every generator.

Disclaimers

Not affiliated

This wiki is not affiliated with, endorsed by, or sponsored by BoltBlasterGames, the developer of The Spell Brigade. All game names, trademarks, and assets belong to their respective owners. This is a fan project, made for the love of the game.

Text-only

We use no game art, audio, or sprite renderings — not even the logo. This keeps the wiki firmly on the safe side of copyright and makes every page load instantly on any connection. The only visual flourishes are CSS-generated badges and a single unicode rune sigil.

Inferred mechanics

Some game mechanics are inferred from data structures in the extracted game data rather than confirmed in-game. These are flagged inline with a callout note block on the relevant page. If you find a discrepancy between what the wiki says and what the game actually does, please assume the game is correct and consider the inference flagged.

Localization keys

In some places you may see localization keys like SpellName_Arrow or DescriptionEntry_Artifact_07 instead of fully localized text. These keys are descriptive enough to identify the item; humanized labels will be added as the wiki improves.

Update cadence

This wiki reflects a snapshot of the current game build. When the developer ships a new build with meaningfully different numbers, the affected pages are regenerated. There is no notification system — this is a snapshot reference, not a live API.

Found an error?

Spot a wrong number, a typo, or a broken link? Open a GitHub issue with a link to the page and a brief description of the problem.

Takedown / removal requests

If you are with BoltBlasterGames (or otherwise hold rights to any content reproduced here) and would like anything removed or modified, please open a GitHub issue on this repo.

Requests are honoured promptly. There is no need to involve lawyers — just file an issue describing what should change, and I will comply within 48 hours of seeing it. This wiki exists to help players and only continues to exist with the publisher's tolerance.
