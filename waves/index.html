Waves

A run plays 20 waves (in a 20-minute window) drawn from a 20-slot timeline. The base wave templates (wave_1 … wave_20) are shared, but each world applies overrides at fixed slots to skew its enemy mix. This index lists every wave entry the game ships — 32 total, including overrides.

Each wave defines: minimum concurrent enemies on field, spawn interval (seconds between spawn events), enemies per spawn event, and a probability table the spawner draws from. Click a wave to drill into its full spawn table.

Elite spawns

Every run spawns a guaranteed elite at 14:00 in-run drawn from EliteBuffer, Spawner, EliteExplodingTick or EliteTeethyWorm. Each stack of StrongerEnemies curse adds an extra elite spawn (at 10:20, 16:20, then 4:10).

Enemy events

Five enemy-event configurations may fire during any wave: Bounce (enemies drop from above), Convergent (spawn at distance, run inward), Line (wall closing in from one direction), Mob (dense cluster at a distance), and the Bounce starter cadence config. See enemy events.
