Artifact 15 — Increase Elemental Damage For Every Infusion

ArtifactType enum: IncreaseElementalDamageForEveryInfusion (id 15). Affects ElementStrengthMultiplier. Spawn rate 50.

Effect

Increase Elemental Damage For Every Infusion modifies the ElementStrengthMultiplier stat. Each infusion-type ImprovementAdded event adds one stack.

The core parameter is 10: Elemental-damage bonus (likely +10% ElementStrengthMultiplier) per infusion-type improvement picked up.

Implemented by the IL2CPP class ArtifactStatModifier_IncreaseElementalDamagePerInfusionPerformed. AddsModifier in the asset YAML is true; spawn rate 50.
