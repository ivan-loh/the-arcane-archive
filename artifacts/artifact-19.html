Artifact 19 — Heal On XP Pickup

ArtifactType enum: HealOnXPPickup (id 19). Affects event-driven (no stat). Spawn rate 50.

Effect

Heal On XP Pickup is event-driven — it does not register a continuous stat modifier. Event: each XP gem pickup rolls a 15% chance to heal for 1 HP.

The core parameter is 15: chanceToHeal — percentage chance (15%) that an XP pickup triggers a heal. Field order in the class is 'chanceToHeal' (0x20) then 'healAmount' (0x24).

Implemented by the compiled class Artifact_HealOnXpPickup. AddsModifier in the asset YAML is false; spawn rate 50.

When to pick

  • High PickupRadius / Luck builds (more XP gems collected).

  • Pairs with XP-gain artifacts (3) for compounding.
