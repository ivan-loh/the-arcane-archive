Damage formula

Formulas are plain monospaced blocks with hand-tagged spans for a light syntax tint — no JS, no preprocessor. 

// Fireball damage at spell level L with artifact stacks A
damage(L, A) = (48 + 6.5 × L) × (1.0 + 0.12 × A)
       × [fire bonus]
       + burn(L) × 3  // 3 burn ticks

Keywords like on_hit, on_kill, and overcharge are inline-highlighted to stand out in prose.

Callouts & keywords

Tip

Holding Shift while picking an upgrade locks the choice, preventing accidental rerolls.

Warning

Acid spells do not stack their DoT on bosses with the resilient trait.

Danger

The Grim Pact doubles incoming damage on Hardcore; never run it without a dedicated healer.

Contributing

This wiki is generated from plain Markdown plus a small set of HTML partials. To add a page, drop a Markdown file under the relevant section folder (spells/, artifacts/, …) and add an entry to data/search-index.json.

  1. Fork the repository.
  2. Run the build script to regenerate html from your edits.
  3. Open a pull request — the maintainers will review for accuracy.
