Game mechanics

This is the “physics manual” of The Spell Brigade. Every page in this section is a direct reference to game systems extracted from the IL2CPP dump: the 36-entry stat enum, the damage pipeline, the 21 element types, the static / dynamic modifier hierarchy, and the defensive math.

Start here

New to the game? Read Stats first to understand what every +25% on an item actually feeds, then jump to Damage system for the full pipeline. Advanced players doing math should bookmark worked examples.

Confidence labels

Game method signatures are extracted directly from the binary, but method bodies are not. Each formula is labelled high (the dump leaves only one reasonable interpretation), inferred (signatures plus a constant strongly imply this), or guess (genre convention). Where confidence is below high, the page says so in a callout — please don’t quote inferred numbers as gospel.
